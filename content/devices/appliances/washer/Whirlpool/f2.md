meaninglong: "The F2 error indicates that the washer’s user interface (UI) has detected a stuck key or unresponsive button. On Whirlpool Cabrio and HE top‑load washers, the UI panel communicates with the main control board through a digital ribbon cable. If a button remains pressed for too long, becomes contaminated with moisture, or if the UI board fails, the washer displays F2 and prevents cycle operation."

  - "Buttons do not respond"

  - "F2 appears immediately after pressing a button"

  - "Stuck or damaged button"
  - "Moisture inside the UI panel"
  - "Loose or corroded ribbon cable"
  - "Failed user interface board"
stepfixes:
  - "Unplug the washer for 5 minutes."
  - "Clean the UI panel and ensure no buttons are stuck."
  - "Remove the console and reseat the ribbon cable."
